# Design — shared package

This package hosts the canonical design system tokens for Amazed Labs.
Two products, one tree:

- **Tono** — professional / dark / violet / lowercase
- **ParentScript** — clinical / light / indigo / sentence case

The token file (`tokens.json`) is the source of truth. Tailwind config
and CSS variables in `apps/web/` mirror it.

See [`/docs/DESIGN-SYSTEM.md`](../../docs/DESIGN-SYSTEM.md) for the
human-readable design system documentation.
